# Contributing

¡Gracias por tu interés en contribuir! 🎉

## Cómo contribuir

1. **Fork** el repositorio
2. **Crea una rama** para tu feature (`git checkout -b feature/amazing-feature`)
3. **Commit** tus cambios (`git commit -m 'feat: add amazing feature'`)
4. **Push** a la rama (`git push origin feature/amazing-feature`)
5. **Abre un Pull Request**

## Estándares

- Escribe tests para nuevo código
- Sigue el estilo existente del código
- Mantén los PRs pequeños y enfocados
- Documenta cambios significativos

## Reportar Issues

Usa el template de issue en GitHub. Incluye:
- Descripción del problema
- Pasos para reproducir
- Comportamiento esperado vs actual
- Screenshots si aplica

## Licencia

Al contribuir, aceptas que tu código será licenciado bajo MIT.
